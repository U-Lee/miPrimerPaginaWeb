Presentación

En estos tiempos de confinamiento provocados por la pandemia de COVID-19 el uso de internet se ha vuelto el principal medio de comunicación, por lo que aprender a usar y crear herramientas de ingeniería de software permitirá adaptarse mejor a esta nueva realidad. La intención de este taller es ofrecer un primer acercamiento al uso de tecnologías web, pero para ello se requiere conocimientos básicos de computación, así que el taller se dividirá en los siguientes módulos y actividades:

Módulo cero. Pioneras, conociendo el ambiente de computación.

Actividad 1: Computadora y sistema
Actividad 2: Expedición por el mundo digital
Actividad 3: La consola

Módulo I. Elaboración de sistemas web.

Taller 1. Construyendo mi primer sitio web

Actividad 1: ¿Qué es una página web?
Actividad 2: Conociendo desde adentro una página web
Actividad 3: Agregando nuevos elementos a tu página web
Actividad 4: “El Konmari” del HTML
Actividad 5: Ponle estilo a tu página
Actividad 6: Utilizando los componentes básicos en del CSS
Actividad 7: De la realidad a la web
Actividad 8: Web de verdad
Actividad 9: De mí para el mundo


En este taller 1 se revisan las tecnologías HTML, CSS y git, con la finalidad de que cuando se finalice, la participante tenga una página web estática que pueda ser consultada de forma remota.

Próximamente se abrirá el Taller 2: Creando un punto de venta en Internet
